# DevSphere 1.0

DevSphere 1.0 is a frontend-only website (HTML/CSS/JS) that presents project info, team members, and a contact form.

## Pages

- Home (`index.html`)
- Project Detail (`detail.html`)
- Contact (`contact.html`)
- Team (`team.html`)

## Buit with

- HTML5
- CSS3
- JavaScript (Vanilla)

## How to Run

Just open `index.html` in a browser.
